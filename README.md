# Capstone: Vector Drawing Application

## Project Structure
- `src/main.cpp`: The entry point and main window manager handling GLUT callbacks.

## Dependencies
- C++ Compiler (GCC, Clang, or MSVC)
- OpenGL
- FreeGLUT (or standard GLUT)

## Compilation
If you are using GCC (MinGW on Windows) with FreeGLUT installed, you compile this via:
```bash
g++ src/main.cpp -o MiniPaint -lfreeglut -lopengl32 -lglu32
```
